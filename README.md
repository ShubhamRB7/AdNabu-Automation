# AdNabu Automation Assignment

## 📌 Project Overview
This project automates the following flow using Selenium:
- Login to AdNabu Test Store
- Open product "The Collection Snowboard: Liquid"
- Add product to cart
- Proceed to checkout

---

## ⚙️ Setup Instructions

### 1. Install Python
Download and install Python 3

### 2. Install Dependencies
Run:
pip install -r requirements.txt

---

## ▶️ How to Run Tests

Run the script using:
python AdNabuFixedTask.py

---

## 📊 Test Result

Expected Output:
- Product is added to cart
- User is redirected to checkout page

If successful:
✅ SUCCESS: Task completed. Item is in the checkout flow.
